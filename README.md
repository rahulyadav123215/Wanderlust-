Wanderlust is a full-stack vacation rental web application inspired by **Airbnb**, allowing users to explore, book, and manage vacation homes. Built using **Node.js, Express, MongoDB, and EJS**, it provides a real-world booking experience with secure authentication, image uploads, and automated email confirmations.
